# Frontend — AI Smart Notes

Next.js app for the AI Smart Notes Manager. For full setup, API docs, and environment variables, see the [root README](../README.md).

## Quick start

```bash
npm install
```

Create `.env.local`:

```env
NEXT_PUBLIC_API_URL=http://localhost:8080
```

```bash
npm run dev
```

Open http://localhost:3000 (backend must be running).

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Development server |
| `npm run build` | Production build |
| `npm run start` | Run production build |
| `npm run lint` | ESLint |
